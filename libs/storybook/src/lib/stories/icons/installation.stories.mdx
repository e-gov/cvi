# How to Install CVI Icons

Latest version: { iconsPkg.version }

1. Add the `@egov` registry to your npm configuration (see the first step in the [general Installation instructions](https://github.com/e-gov/cvi#%EF%B8%8F-installation-instructions)).

2. Install the library:
```
npm i --save @egov/cvi-icons
```

1. Check the [usage page](?path=/docs/icons-how-to-use--page) for integration examples. Remember that for Angular, integration includes registering an icon in an icons registry.
