# How to Install the Angular Diagrams library

Latest version: { diagramsPkg.version }

## peerDependencies

```
"@angular/animations": ">=14.1.0",
"@angular/common": ">=14.1.0",
"@angular/core": ">=14.1.0",
"@angular/platform-browser": ">=14.1.0",
"@angular/platform-browser-dynamic": ">=14.1.0",
```

## Installation

1. Add the `@egov` registry to your npm configuration (see the first step in the [general Installation instructions](https://github.com/e-gov/cvi#%EF%B8%8F-installation-instructions)).

2. Install the library
```
npm i --save @egov/cvi-ng-diagrams
```

3. Import `NgDiagramsModule` in your `app.module.ts`.

```ts

import { NgDiagramsModule } from '@egov/cvi-ng-diagrams';

@NgModule({
  ...
  imports: [NgDiagramsModule],
  ...
})
```
