## Installation

1. Install the library
```
npm i --save @egov/cvi-ng-diagrams
```

2. Import `NgDiagramsModule` in your `app.module.ts`.

```ts

import { NgDiagramsModule } from '@egov/cvi-ng-diagrams';

@NgModule({
  ...
  imports: [NgDiagramsModule],
  ...
})
```
