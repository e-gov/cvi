# Spacing patterns

CVI has numerous options when it comes to spacing.

Spacing patterns are named according to a [list of Estonian cities](https://en.wikipedia.org/wiki/List_of_cities_and_towns_in_Estonia) and sorted by their population. This should give a good idea about a pattern's size while allowing to insert newer sizes in between.

You can pick a font size by employing Sass variables (with `get-spacing()` function) or using `--cvi-spacing-` CSS variables.

<Story name="Spacing" height="650px">

      </cvi-ng-storybook-spacing-specimen>
    `,
  }}
</Story>
