# Colors

CVI has a choice of colors grouped by basic color variations.

Colors can be used both in Sass (`get-color(dark-tangerine-0)`) and as CSS variables (`var(--cvi-color-dark-tangerine-0)`);

## Black coral

<Story name="Black coral" height="540px">
  {{
    props: { colorsTotal },
    template: `
      <cvi-ng-storybook-color-cards>
        <cvi-ng-storybook-color-card *ngFor="let _ of [].constructor(colorsTotal); let i = index" label="black-coral-{{ i }}" [ngStyle]="{'--color': 'var(--cvi-color-black-coral-' + i + ')'}"></cvi-ng-storybook-color-card>
      </cvi-ng-storybook-color-cards>
    `,
  }}
</Story>
