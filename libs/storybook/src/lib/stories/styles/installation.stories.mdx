# How to Install CVI Styles

Latest version: { stylesPkg.version }

The library styles are built using Sass (namely [Dart Sass](https://sass-lang.com/dart-sass)).

1. Add the `@egov` registry to your npm configuration (see the first step in the [general Installation instructions](https://github.com/e-gov/cvi#%EF%B8%8F-installation-instructions)).

2. Install the library
    ```
    npm i --save @egov/cvi-styles
    ```
3. Include styles to a Sass file in the project:
    ```
    @use '@egov/cvi-styles/main';
    ```

You can also `@use` or `@import` a specific Sass module from the library, eg. `@use '@egov/cvi-styles/settings/variables/typography';`.
