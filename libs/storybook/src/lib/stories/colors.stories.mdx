# Colors

See [information on colors on the Veera Design System page](https://veera.eesti.ee/3d136290e/p/820875-vrvid/b/845dc1).

Veera has a choice of colors grouped by basic color variations.

Colors can be used both in Sass (`get-color(dark-tangerine-0)`) and as CSS variables (`var(--veera-color-dark-tangerine-0)`);

## Black coral

<Story name="Black coral" height="540px">
  {{
    props: { colorsTotal },
    template: `
      <veera-ng-storybook-color-cards>
        <veera-ng-storybook-color-card *ngFor="let _ of [].constructor(colorsTotal); let i = index" label="black-coral-{{ i }}" [ngStyle]="{'--color': 'var(--veera-color-black-coral-' + i}"></veera-ng-storybook-color-card>
      </veera-ng-storybook-color-cards>
    `,
  }}
</Story>
