# How to Install the Angular UI Kit

Latest version: { uiPkg.version }

## peerDependencies

```
"@angular/animations": "^14.1.1",
"@angular/cdk": "^14.2.0",
"@angular/common": "^14.1.1",
"@angular/core": "^14.1.1",
"@angular/elements": "^14.1.3",
"@angular/platform-browser": "^14.1.1",
"@angular/platform-browser-dynamic": "^14.1.1"
```

## Installation

1. Add the `@egov` registry to your npm configuration (see the first step in the [general Installation instructions](https://github.com/e-gov/cvi#%EF%B8%8F-installation-instructions)).

2. Install the library
```
npm i --save @egov/cvi-ng
```

3. Import `UiModule` in your `app.module.ts`.

```ts
import { UiModule } from '@egov/cvi-ng';

@NgModule({
  ...
  imports: [BrowserModule, UiModule],
  ...
})
```

You may want to take a look at an example app at [codesandbox.io](https://codesandbox.io/p/github/ekateriinal/angular-cvi-starter).
