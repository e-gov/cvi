## Installation

1. Install the library
```
npm i --save @ria/veera-ng;
```

2. Import UiModule in your app.module.ts.

```
import { UiModule } from '@ria/veera-ng';

@NgModule({
  ...
  imports: [BrowserModule, UiModule],
  ...
})
```
