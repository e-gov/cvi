## General description

| Feature                    | Description                                                          |
| -------------------------- | -------------------------------------------------------------------- |
| Selector                   | `cvi-ng-icon`                                                      |
| Component name             | `IconComponent`                                                      |
| Content                    | no                                                                   |
| Permitted parent/ancestors | any                                                                  |
| Permitted children         | -                                                                    |

## Where to use

Displays a regular svg icon.

Please remember to register the icon via `IconsRegistry` in your app.
